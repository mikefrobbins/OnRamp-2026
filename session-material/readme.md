# Session Materials

This folder is used by OnRamp presenters to share session content with participants.

Each presenter should upload their materials in a dedicated subfolder named after their session.
This helps keep content organized and easy to find.

Examples:

```
session-material/
├── advanced-functions/
├── remoting-at-scale/
```

## What to include

Presenters may upload any of the following:

- Slide decks (PDF or PowerPoint)
- Demo scripts
- Sample code
- Lab instructions
- Supporting documents or links

Please ensure files are clearly named and organized.

## Guidelines

- Use descriptive folder and file names.
- Avoid uploading large binaries unless necessary.
- Do not include credentials, secrets, or sensitive information.
- Test scripts before uploading.
- Prefer Markdown for written instructions when possible.

## For participants

Session content will appear here as presenters upload materials throughout the week.

If something is missing, check back later or ask in the OnRamp Slack channel.

Thank you to all presenters for contributing and helping make OnRamp a great experience.
