# OnRamp to Success: A Pathway for Aspiring IT Professionals

<img align="right" src="../images/powershell-hero.jpg" alt="PowerShell hero image." >

Welcome to **OnRamp**, a structured introduction to PowerShell and modern IT automation for aspiring
IT professionals and those strengthening foundational automation skills. The program combines
instructor-led sessions, demonstrations, guided exercises, and hands-on labs to teach real-world
PowerShell, automation, and DevOps concepts.

## What is OnRamp?

OnRamp is a multi-day training program delivered as a dedicated Summit track. It focuses on core
PowerShell concepts and applied learning aligned with modern IT automation workflows.

Participants follow a structured, hands-on agenda covering PowerShell fundamentals, scripting,
functions, modules, automation, source control, and labs. Over four days, you’ll learn how to write,
test, debug, and share code while reinforcing those skills through guided exercises.

As a _conference-within-a-conference_ at the PowerShell + DevOps Global Summit, OnRamp provides
full-day training while incorporating select Summit keynotes and events. Attendees move through the
program together, with shared breaks and meals that support networking and collaboration.

**Participation in all OnRamp sessions is required.** Attendees remain within the OnRamp track and
do **not** switch between OnRamp and main conference sessions.

## Week at a glance

**By the end of OnRamp, you will be able to:**

- Install and configure PowerShell 7 on Windows, Linux, or macOS
- Discover commands, understand syntax, and use the pipeline effectively
- Write scripts, build advanced functions, and create script modules
- Debug PowerShell code using structured troubleshooting techniques
- Automate across systems using remoting and reusable tooling
- Use Git and GitHub to manage, share, and contribute PowerShell code

## Program highlights

- **Structured, hands-on curriculum:** Progresses from PowerShell basics to scripting, testing,
  automation, and real-world scenarios.
- **Instruction from experienced practitioners:** Learn directly from professionals actively working
  in PowerShell and automation.
- **Community and networking:** Build connections with peers, instructors, and the broader Summit
